## 世界観の根幹

### この世界の設定

日本の霊的世界には「見えない守護の秩序」が存在する。

人間が生まれると同時に、その魂の性質と深さに呼応した
**守護存在**が引き寄せられ、生涯を通じて寄り添うとされる。

守護存在たちは **四つの階（かい）** に分かれており、
それぞれ異なる起源・使命・力の性質を持つ。

彼らは人間に直接姿を見せることはない。
しかし──「気配」として、「夢」として、
あるいは「直感」として、確かにその存在を伝えてくる。

---

### 四階層の世界構造

```
【第一階】神獣（しんじゅう）
  宇宙の法則と自然の力そのものを体現した原初の存在。
  人間界の願いを超えた次元で動き、
  その者の魂の「格」に応じてのみ力を貸す。

      ↕ 霊的なつながり

【第二階】神使（かみつかい）
  神々の意志を人間界に伝える媒介者たち。
  縁・富・道案内など、具体的な恩恵を「運んでくる」存在。
  神と人の間に立ち、双方向のパイプとなる。

      ↕ 境界を越える者たち

【第三階】妖異（ようい）
  人間と霊的世界の境界から生まれた、いずれにも属さない存在。
  深く強く、時に危うい力を持ち、その者の本質を試し、
  真の実力を引き出す役割を担う。

      ↕ 人に最も近い守護

【第四階】守護霊（しゅごれい）
  人の感情・記憶・土地・時間から生まれた霊的存在。
  四階層の中で最も人間に近く、日常の中に静かに宿る。
```

---

## 四階層の詳細設定

### 【第一階】神獣（しんじゅう）

> 「この世界が始まる前から、彼らはいた」

自然法則そのものが神格化された存在。
時間も空間も超えており、人間一人の事情よりも
「世界全体の流れ」を優先して動く。

神獣に守護される者は、個人の枠を超えた使命を持つとされる。
普段は遠くにいるが、人生の「決定的な瞬間」に力を貸してくる。

**階層の特性：**
- 守護力：圧倒的・圧倒的に広い範囲
- 関わり方：非日常的・転機の瞬間に力を貸す
- 試練の性質：「大きな流れへの抵抗」を求められる
- カードのレアリティ（想定）：SSR

**属する守護存在：龍神 ／ 鳳凰 ／ 麒麟**

---

### 【第二階】神使（かみつかい）

> 「彼らは常に動いている。縁の糸を、あなたのために」

神々に仕え、人間界との橋渡しをする存在。
「何かが降ってくる」感覚──突然のチャンス、不思議な出会い、
夢の中の啓示──それは神使が動いている証とされる。

神使に守護される者は、縁・情報・タイミングに恵まれやすい。
ただし「受け取る側の姿勢」が問われる。

**階層の特性：**
- 守護力：具体的・縁と機会に強い
- 関わり方：日常に近く、タイミングとして現れる
- 試練の性質：「受け取る勇気があるか」を問われる
- カードのレアリティ（想定）：SR

**属する守護存在：稲荷狐 ／ 八咫烏 ／ 白蛇**

---

### 【第三階】妖異（ようい）

> 「彼らの守護は優しくない。しかし、最も本物だ」

人間と霊的世界の「狭間」に生まれた存在。
長い年月と強烈な経験を経て自らの力を得た、
人間でも神でもない、第三の存在たち。

妖異に守護される者は、深い試練と共に圧倒的な才能を得る。
「生きやすさ」よりも「深さ」を与えられる守護者たち。

**階層の特性：**
- 守護力：個人の才能・能力の覚醒に特化
- 関わり方：内側から──直感・夢・感性として現れる
- 試練の性質：「自分の本質から逃げないこと」を求められる
- カードのレアリティ（想定）：R（だが人気が高い）

**属する守護存在：九尾の狐 ／ 猫又 ／ 天狗**

---

### 【第四階】守護霊（しゅごれい）

> 「彼らはあなたの隣に、今も、いる」

人の喜び・悲しみ・祈り・土地の記憶から生まれた霊的存在。
最も身近で、最も温かく、最も「人間の事情」を理解している守護者。

守護霊に守護される者は、日常の小さな幸せと縁に守られる。
大きな力ではなく、「毎日をちゃんと生きる力」を与えてくれる。

**階層の特性：**
- 守護力：日常・感情・家庭・縁に特化
- 関わり方：最も日常的──場の空気・体の感覚として現れる
- 試練の性質：「今いる場所・今の自分を大切にすること」を問われる
- カードのレアリティ（想定）：N〜R（だが守護霊は"外れなし"の設定）

**属する守護存在：雪女 ／ 座敷童子 ／ 護り鬼**

#### 01. 龍神（りゅうじん）

**二つ名：「深淵の主宰者」**

```
階級   ：神獣（第一階）
守護属性：水・天・豊穣
象徴紋  ：渦潮の中心から昇る龍（波紋円に昇龍が絡む構図）

         生命の根源となる水の力で万物に豊かさをもたらすこと
```

**得意な人生テーマ**
| テーマ | 特性 |
|---|---|
| ◎ 金運・豊かさ | 流れを呼び込む最高の守護 |
| ◎ 大きな転機 | 人生の大波を引き寄せ乗りこなす力 |
| ○ 仕事・事業 | 拡大と流れを動かす局面に強い |
| △ 恋愛 | 感情の波が激しくなることもある |

**才能の核：大局を読む力、豊かさを引き寄せる引力、場の流れを変えるカリスマ**

**弱点の核：感情の波、継続より爆発型、境界線の曖昧さ**

**開運行動（3つの核心）**
1. 水のそばに立つ。海・川・湖──流れを体で感じる
2. 龍神社へ参拝し「流れをください」と願う
3. 深藍・金のアイテムを財に関わるものに取り入れる

**相性：◎ 八咫烏（天と海、二つの極が一つに）／ ○ 鳳凰**

### 象徴紋の活用方針

象徴紋はこのIPの最重要視覚資産。
以下の用途に統一して展開する。

```
デジタルお守り → 象徴紋 + 守護存在名のシンプルデザイン
カード      → 象徴紋を背景・透かしに使用
壁紙       → 象徴紋 + 守護存在イラストの合成
PDF表紙     → 象徴紋を大きく配置
物販       → 象徴紋をメインモチーフに
```

### イラスト制作ガイドライン（発注時参照）

#### 全体スタイル
- 和風×神秘的×モダン（水墨画ベース ＋ 金彩）
- 性別：守護存在は基本的に性別を感じさせない（ただし鳳凰・雪女は女性的に）
- 背景：各存在の守護属性に合わせたグラデーション

#### 階級別の描き分け
```
神獣     → 壮大・圧倒的なスケール感。画面全体に圧がある
神使     → 動的・軽やか・存在が光を持つ
妖異     → 神秘的・アンビバレント・影と光が共存
守護霊   → 温かみ・親しみ・日常に溶け込む存在感
```

### 世界観テキストの設計（全商品共通の「口調」）

```
NG：占い・スピリチュアル業界の一般的な文体
  「あなたの運気は〜」「今年の運勢は〜」

OK：守護存在が語りかける、詩的で物語的な文体
  「あなたの中に、深淵の水が流れている。
   それを力と呼ぶ者もいる。
   ただ、あなただけが知っている。
   その水は、氾濫するためにあるのではない。」
```

守護存在の語りかけは常に「一人称×詩的×短文」で統一。
これがブランドの声（Brand Voice）になる。
