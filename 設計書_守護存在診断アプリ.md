# 日本の守護存在診断アプリ 設計書

作成日：2026-06-15

---

## 1. アプリ全体のコンセプト

### タイトル案
- 「守護存在診断 ― あなたを護る神獣・霊獣・妖怪を知る」
- 「和の守護者診断」

### コンセプト
生年月日を入力するだけで、日本古来の守護存在（神獣・霊獣・神使・妖怪）の中から
あなたの守護者を導き出す占いWebアプリ。

数秘術（ライフパスナンバー）を診断エンジンに使い、
結果には守護存在のビジュアル・性格分析・ラッキー属性を表示する。

### ターゲットユーザー
- 占い・スピリチュアルに関心がある10〜40代女性（メイン）
- 日本神話・妖怪・和風ファンタジー好き
- 自己分析・自己理解に興味がある層

### デザイン方針
- 和風×神秘的：墨絵・金箔・和紙テクスチャーをベースにしたUI
- 暗めの背景（深藍・漆黒）に金・朱・翠の差し色
- 診断中は水墨画エフェクトで守護存在が浮かび上がる演出
- モバイルファーストのレスポンシブ設計

## 2. 診断ロジック（数秘術）

### ライフパスナンバーの計算方法

生年月日の各桁をすべて足し合わせ、1桁になるまで繰り返す。
ただし途中で **11 / 22 / 33** が出た場合はマスターナンバーとして止める。

```
例：1990年5月15日生まれ

YYYY  MM  DD
1990  05  15

各桁の合計：1+9+9+0+0+5+1+5 = 30
           3+0 = 3

→ ライフパスナンバー：3
```

```
例：1985年11月29日生まれ

1+9+8+5+1+1+2+9 = 36
3+6 = 9

→ ライフパスナンバー：9
```

```
例：マスターナンバーのケース
1+9+7+4+0+2+0+2 = 25
2+5 = 7 → そのまま7

1+9+8+8+1+1+2+9 = 39
3+9 = 12 → 1+2 = 3

マスターナンバーとなるケース（途中で11/22/33が出る場合）：
計算途中で合計が29 → 2+9=11 → ここで止めて「11」とする
計算途中で合計が22 → そのまま「22」とする
計算途中で合計が33 → そのまま「33」とする
```

## 3. 守護存在12体の候補

### No.1 ── 青龍（せいりゅう）
東天を司る四神の一柱。蒼く輝く鱗を持つ巨大な龍。
春の生命力と東の方角を守護し、始まりと成長をもたらす。

### No.2 ── 朱雀（すざく）
南天を司る四神の一柱。炎の翼を持つ不死の霊鳥。
情熱と創造のエネルギーを持ち、縁結びと芸術的才能を高める。

### No.3 ── 白虎（びゃっこ）
西天を司る四神の一柱。純白の虎。
勇気と決断力の象徴。邪を払い、切り開く力を与える。

### No.4 ── 玄武（げんぶ）
北天を司る四神の一柱。亀と蛇が一体となった神獣。
忍耐・安定・守護の力が最も強い。長寿と財の守護者。

### No.5 ── 麒麟（きりん）
龍頭・鹿体・牛蹄・鰭尾を持つ幸運の霊獣。
仁徳ある者の前にのみ姿を現すとされ、自由と変革の象徴。

### No.6 ── 鳳凰（ほうおう）
百鳥の王。桐の木にしか止まらず、竹の実のみを食べる高貴な霊鳥。
再生・調和・家族の繁栄をもたらす。

### No.7 ── 八咫烏（やたがらす）
天照大神の使いである三本足の霊烏。神武天皇を熊野から大和へ導いた。
導き・勝利・神託の象徴。迷いを断ち切り正道へ導く。

### No.8 ── 龍神（りゅうじん）
海と水を司る神。竜宮城の主。
豊穣・財運・潮流を支配し、大きな流れに乗る力を与える。

### No.9 ── 稲荷狐（いなりぎつね）
稲荷大神に仕える神使の白狐。全国最多の神社を持つ稲荷信仰の象徴。
商売繁盛・五穀豊穣・縁結びを司る。知恵と霊力が際立って高い。

### No.11 ── 天狗（てんぐ）
山岳に宿る高位の霊的存在。修験道と深く結びつく。
傲慢を戒め、武芸・霊力・深山の秘術を授ける。試練を通じて覚醒させる守護者。

### No.22 ── 白蛇（しろへび）
弁財天・宇賀神の神使。白蛇は金運・財運の最高の象徴とされる。
静かに、しかし確実に富と知恵を蓄える力を与える。

### No.33 ── 不知火（しらぬい）
海上に現れる神秘の炎。有明海・橘湾に伝わる霊的現象の神格化。
超感覚・霊感・人知を超えた導きの象徴。最も稀少なマスター守護存在。

## 5. 結果ページの構成

### ページフロー

```
[TOPページ]
  ↓ 生年月日入力
[診断中演出ページ]  ← 3〜5秒のローディングアニメーション
  ↓
[結果ページ]
  ↓
[詳細鑑定書CTA] / [SNSシェア]
```

### TOPページ

- タイトル・キャッチコピー
- 生年月日入力フォーム（年・月・日プルダウンまたはカレンダー）
- 「診断する」ボタン
- 簡単な説明文（数秘術とは何か、守護存在とは何か）

### 診断中演出ページ

- 和紙テクスチャ背景に墨が広がるアニメーション
- 「守護存在を呼び覚ましています...」などのテキスト演出
- 計算処理は即時完了するが、演出のため3〜5秒表示

### 結果ページ（メイン）

```
─────────────────────────────────────────
① ヘッダー（守護存在名の大きな表示）
   「あなたの守護存在は──」
   守護存在名（大）／ふりがな／種別バッジ

② ビジュアルエリア
   守護存在のイラスト（和風・墨絵調）
   背景に五行カラーのグラデーション

③ 守護のメッセージ
   守護存在からの一言メッセージ（詩的な文体）

④ あなたの性格分析
   3〜4つの特性をカード形式で表示
   例：「開拓者」「直感力」「意志の強さ」

⑤ 守護属性パネル
   ┌───────┬───────┬───────┐
   │ 五行  │ 方位  │ 季節  │
   │ ラッキーカラー │ ラッキーアイテム │
   └───────────────────────┘

⑥ 相性の守護存在
   相性の良い守護存在のミニカード表示
   「〇〇の守護者を持つ人と縁がある」

⑦ SNSシェアボタン
   X（Twitter）/ LINE / コピー
   シェアテキスト自動生成
   OGP画像で守護存在ビジュアルが表示される

⑧ 販売導線CTA（詳細は後述）
─────────────────────────────────────────
```
